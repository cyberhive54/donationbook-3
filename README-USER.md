## Getting Started

### Step 1: Get Your Festival Code

**If your festival already exists:**
1. Ask your festival organizer for the 8-letter festival code
2. Example: `RHSPVM25`

**To create a new festival:**
1. Visit the Donation Book website
2. Click "Create New Festival"
3. Fill in festival details:
   - Festival name (e.g., "Diwali 2026")
   - Organizer name
   - Start and end dates
   - Location
4. Set passwords:
   - **Super Admin Password** (for you, the creator)
   - **Admin Password** (for other managers - optional)
   - **Visitor Password** (for general viewers - optional)
5. Click "Create Festival"
6. Save your unique festival code!

### Step 2: Access Your Festival

1. Go to the Donation Book website
2. Enter your festival code
3. Enter your name and password (if required)
4. Click "Continue"

That's it! You're in! üéâ

## User Roles Explained

### üë§ Visitor Access

## How to Use

### For Visitors

#### View Festival Dashboard
After logging in, you'll see:
- Festival banner with key information
- Four statistics cards:
  - Total Collection
  - Total Expense
  - Total Balance
  - Unique Donors
- Recent transactions (latest 10)
- Navigation bar at the bottom

#### Browse Collections
1. Click "Collections" in bottom navigation
2. View all donations in a table
3. Sort by clicking column headers
4. Use search to find specific donors
5. Export to CSV if needed

#### Browse Expenses
1. Click "Expenses" in bottom navigation
2. View all expenses in a table
3. Sort and filter by category
4. See item details and quantities

#### View Analytics
1. Click "Analytics" in bottom navigation
2. Explore interactive charts:
   - Collection vs Expense trends
   - Top donors
   - Donation by groups
   - Expense by categories
   - Time-based analysis

#### Browse Gallery
1. Click "Showcase" in bottom navigation
2. Select an album
3. View photos, videos, and media
4. Click on any item to view full size
5. Download if allowed

---

### For Admins

#### Add a Collection (Donation)
1. Login as Admin
2. Go to "Collections" section
3. Click "Add Collection"
4. Fill in details:
   - Donor name
   - Amount
   - Group (select or create new)
   - Payment mode
   - Date and time
   - Optional note
5. Click "Save"

#### Add an Expense
1. Go to "Expenses" section
2. Click "Add Expense"
3. Fill in details:
   - Item name
   - Number of pieces
   - Price per piece (total calculated automatically)
   - Category
   - Payment mode
   - Date and time
   - Optional note
4. Click "Save"

#### Manage Visitor Passwords
1. In Admin Dashboard, find "User Password Management"
2. Click "Manage Passwords"
3. **To Add Password:**
   - Click "Add New Password"
   - Enter password (or use generated one)
   - Add label (e.g., "Committee Members")
   - Click "Save"
4. **To Edit Password:**
   - Click edit icon
   - Modify password or label
   - Click "Save"
5. **To Deactivate Password:**
   - Toggle the "Active" switch
   - Users with this password will be logged out in 5 minutes
6. **To View Usage:**
   - Click "View Usage" to see who used the password

#### Upload Media
1. Go to "Showcase" section
2. Click "Manage Albums"
3. **To Create Album:**
   - Click "Add Album"
   - Enter title, description, year
   - Upload cover image
   - Set download permissions
   - Click "Save"
4. **To Add Media:**
   - Open an album
   - Click "Upload Media"
   - Choose files or add external links
   - Add titles and descriptions
   - Click "Upload"

---

### For Super Admins

#### Create New Admin
1. Login as Super Admin
2. Go to "Admin Management" section
3. Click "Create Admin"
4. Fill in details:
   - Admin code (e.g., "ADMIN02") or auto-generate
   - Admin name (e.g., "Rahul Kumar")
   - Password
   - Max user passwords (1-10)
5. Check "Active" checkbox
6. Click "Create Admin"

#### Edit Festival Settings
1. Click "Edit Festival Settings"
2. Modify:
   - Festival name
   - Festival code
   - Dates (event dates, collection/expense dates)
   - Location, organizer, guide, mentor
3. Click "Save Changes"
4. **Note**: Changing festival code will require re-login

#### Customize Theme
1. Scroll to "Theme Settings"
2. Click "Edit Theme"
3. Customize:
   - Primary and secondary colors
   - Background color
   - Text and border colors
   - Background image URL
   - Dark mode toggle
4. Click "Save Theme"
5. See changes immediately!

#### Configure Analytics
1. Click "Configure Analytics"
2. Set:
   - Collection target amount
   - Target visibility (public or admin-only)
   - Previous year data (collection, expense, balance)
   - Donation amount buckets (ranges)
   - Time-of-day buckets (hour ranges)
3. Select which analytics cards to display
4. Click "Save Configuration"

#### View Activity Logs
1. Navigate to "Activity" page
2. See complete history of:
   - Who added/edited/deleted what
   - When it happened
   - Admin name and action details
3. Filter by admin or date
4. Export for record-keeping

## Features Guide

### üìä Statistics Cards

Displayed on every page for quick overview:

1. **Total Collection**: Sum of all donations
2. **Total Expense**: Sum of all expenses
3. **Balance**: Collection minus Expense
4. **Unique Donors**: Number of different donor names

### üìÖ Date Management

**Two Date Ranges:**

1. **Collection/Expense (CE) Dates** (Required)
   - The period when donations and expenses can be recorded
   - All transactions must fall within this range

2. **Event Dates** (Optional)
   - The actual festival dates
   - Must be within CE dates
   - Displayed on festival banner

**Important**: You cannot add transactions outside the CE date range!

### üîê Password Management

#### Visitor Passwords
- Each admin can create up to 10 visitor passwords
- Each password has a label (e.g., "Building A Residents")
- Track usage count and last used date
- See which visitors used which password
- Deactivate to force logout users

#### Password Features
- **Name Pre-fill**: Your name is remembered on your device
- **Concurrent Session Detection**: Warns if you're already logged in
- **Auto-logout**: Sessions expire daily at midnight (IST)
- **Force Logout**: Admin can deactivate password to logout users

### üìà Analytics Features

## Frequently Asked Questions

### General Questions

**Q: Is Donation Book free to use?**  
A: Yes! Donation Book is completely free for communities and organizations.

**Q: Do I need to install anything?**  
A: No, it's a web application. Just open the URL in any browser.

**Q: Can I use it on my phone?**  
A: Absolutely! It works on phones, tablets, and computers.

**Q: How many festivals can I create?**  
A: Unlimited! Each festival has a unique code for complete isolation.

**Q: Is my data safe?**  
A: Yes, all data is stored securely in a professional database with password protection.

---

### Festival Creation

**Q: Can I change the festival code after creation?**  
A: Yes, super admins can change it in "Edit Festival Settings". Everyone will need to use the new code.

**Q: What if I forget my super admin password?**  
A: Currently, passwords cannot be recovered. Make sure to save it securely!

**Q: Can I create a festival without passwords?**  
A: Visitor password is optional. You can disable "Requires Password" to allow free access for visitors. However, Admin and Super Admin passwords are mandatory.

**Q: What's the difference between event dates and CE dates?**  
A: CE dates define when transactions can be recorded (wider range). Event dates are the actual festival dates (must be within CE dates).

---

### User Management

**Q: How many admins can I have?**  
A: Unlimited admins per festival!

**Q: Can an admin create other admins?**  
A: No, only super admins can create/manage admins.

**Q: What happens if I deactivate an admin?**  
A: They are immediately logged out and cannot login until reactivated.

**Q: How many visitor passwords can each admin create?**  
A: Each admin can create 1-10 passwords (configurable by super admin when creating the admin).

**Q: Can two visitors use the same password?**  
A: Yes, but each must use a unique name. Names are case-insensitive and must be unique per festival.

**Q: Can I see who is currently logged in?**  
A: You can see access logs showing who logged in when, and session duration.

---

### Collections & Expenses

**Q: Can I edit a collection after adding it?**  
A: Yes, admins can edit any collection from the admin dashboard.

**Q: Can I add collections for past dates?**  
A: Yes, as long as the date is within the CE date range.

**Q: What if I add a transaction outside CE dates?**  
A: The system will prevent it and show an error. You can view out-of-range transactions in the admin panel.

**Q: Can I delete transactions?**  
A: Yes, admins and super admins can delete collections and expenses.

**Q: Who can see who added a transaction?**  
A: The system tracks which admin created/updated each transaction. Super admins can view this in activity logs.

---

### Media & Showcase

**Q: What's the maximum file size for uploads?**  
A: Configurable by super admin. Default is 100 MB for videos, 50 MB for other files.

**Q: Can I link Google Drive videos instead of uploading?**  
A: Yes! Use the "Add External Link" option when adding media.

**Q: Who can upload media?**  
A: All admins and super admins.

**Q: Can I disable downloads for certain albums?**  
A: Yes, set "Allow Download" to false when creating/editing an album.

**Q: What happens if I exceed storage limits?**  
A: New uploads will be rejected. Contact your super admin to increase limits or delete old media.

---

### Technical Questions

**Q: Which browsers are supported?**  
A: All modern browsers: Chrome, Firefox, Safari, Edge, Opera.

**Q: Does it work offline?**  
A: No, you need an internet connection to access data.

**Q: Can I export my data?**  
A: Yes, collections and expenses can be exported to CSV and JSON formats.

**Q: Can I import data from Excel?**  
A: Yes, you can import from JSON format (Excel can be converted to JSON).

**Q: What timezone is used?**  
A: Indian Standard Time (IST, UTC+5:30) for session management. Dates are stored as-is.

## Tips & Best Practices

1. **Save Your Passwords**: Write down super admin password securely
2. **Regular Backups**: Export data periodically as CSV
3. **Test Before Festival**: Create test collections/expenses to familiarize yourself

5. **Monitor Activity**: Check activity logs regularly
6. **Set Clear Labels**: Use descriptive labels for user passwords

### For Admins

1. **Accurate Data Entry**: Double-check amounts and names before saving
2. **Use Groups Effectively**: Organize donors by building, committee, etc.
3. **Consistent Categories**: Use same category names for similar expenses
4. **Add Notes**: Include useful context in the note field
5. **Upload Receipts**: Take photos of receipts and attach to expenses
6. **Regular Updates**: Enter transactions daily, don't wait until the end

### For Visitors

1. **Bookmark the URL**: Save festival URL for easy access
2. **Check Regularly**: Stay updated with latest collections and expenses
3. **Explore Analytics**: Use charts to understand donation patterns
4. **Suggest Improvements**: Share feedback with your admins

## Glossary

- **Festival Code**: Unique 8-letter identifier for each festival (e.g., RHSPVM25)
- **CE Dates**: Collection/Expense date range when transactions can be recorded
- **Visitor**: Regular user with view-only access
- **Admin**: User who can add/edit/delete collections and expenses
- **Super Admin**: Festival creator with full control
- **Session**: Your logged-in period (expires daily at midnight IST)
- **Device ID**: Unique identifier for your device (used for name pre-fill)
- **RPC**: Remote Procedure Call (database function for login/logging)
- **RLS**: Row Level Security (database security feature)
